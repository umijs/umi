> Example for building frontend app with [umi](https://github.com/umijs/umi) Server-Side Render

### 🏠 [Homepage](https://github.com/umijs/umi)

## Prerequisites

- node &gt;=8.9.0

## Install

```sh
$ npm install
```

## Usage

Development

```sh
$ npm run dev
$ open http://localhost:7001/
```

Start server

```sh
$ npm start
```

Pre Render

```sh
$ npm run prerender
```
