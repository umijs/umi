# umi-plugin-react

## Usage

Install via yarn or npm.

```bash
$ yarn add umi-plugin-react
```

Configure it in the `.umirc.js`.

```js
export default {
  plugins: [
    ['umi-plugin-react', option],
  ],
};
```
