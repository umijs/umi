# babel-preset-umi

Yet another babel preset.

## Options

### browsers

Notice: browsers don't work if targets is specified.

```js
["umi", {
  "browsers": ['last 2 versions']
}]
```

### targets

e.g.

```js
["umi", {
  "targets": {
    "browsers": ['last 2 versions']
  }
}]
```

or

```js
["umi", {
  "targets": {
    "node": "6"
  }
}]
```
