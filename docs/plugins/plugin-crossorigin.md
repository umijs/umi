# @umijs/plugin-crossorigin

Add the `crossorigin="anonymous"` attribute to all non-third party scripts, which is usually used to count script errors.

## How to enable

The configuration is turned on.

## Configuration

* Type: `boolean`
* Default: `false`

such as:

```js
export default {
  crossorigin: true,
}
```
