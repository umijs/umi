## 三分钟上手 Umi

手动创建文件，

```bash
# 创建目录
$ mkdir myapp && cd myapp

# 安装依赖
$ yarn add umi

# 创建页面
$ npx umi g page index --typescript --less

# 启动开发
$ npx umi dev
```

或者[通过脚手架快速上手](/zh-CN/docs/getting-started)。
